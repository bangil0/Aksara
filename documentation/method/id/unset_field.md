unset_field

Metode ini untuk menghilangkan atau menghidden suatu kolom pada inputan

###### Referensi

`unset_field($params)`

###### Parameter
Parameter
* **$params** (mixed) - nama daripada field

###### Contoh Penggunaan

```php
$this->unset_field
(
	array
	(
		'judul',
		'author'
	)
);
```
